**Title:** ConduitScore Scans Websites for AI Visibility — 85% Missing Key Signal Files

**Body (for submission):**
A new tool called ConduitScore (conduitscore.com) scans websites and returns a 0-100 score measuring how visible a site is to AI agents like ChatGPT, Claude, and Perplexity. The tool checks seven signal categories including robots.txt AI bot permissions, llms.txt file presence, JSON-LD structured data, author citation signals, content structure, and technical health. After scanning approximately 500 production websites, the tool found that 85% lack llms.txt files and 40% have robots.txt configurations that block AI crawlers. The free tier allows 3 scans/month; paid tiers unlock all code-level fixes and scheduled monitoring.
