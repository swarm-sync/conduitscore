# freeCodeCamp — Submission Guide

## PREREQUISITE WARNING
freeCodeCamp requires 3+ published articles on reputable publications before accepting submissions. Verify you qualify before submitting.

## Submission Steps
1. Submit via: https://www.freecodecamp.org/news/contributing/
2. Email: editorial@freecodecamp.org  
3. Pitch the article first before submitting the full draft
4. Suggested pitch subject: "Pitch: Technical guide to AI crawler visibility for web developers"
5. Include: 2-3 sentence summary, your published writing samples, your technical background
6. If accepted, submit the full article.md draft
7. freeCodeCamp editorial team will edit for style — expect minor changes

## Notes
- freeCodeCamp audience is developers — the technical depth in this article is appropriate
- The checklist format is ideal for freeCodeCamp readers (skim → save → reference)
- Avoid promotional tone — the single tool mention at the end is appropriate; more would not be
