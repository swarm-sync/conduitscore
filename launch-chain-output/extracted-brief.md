Product: ConduitScore
Tagline: AI Visibility Score Scanner — crawl any website and score its visibility to AI agents (ChatGPT, Claude, Perplexity)
Category: SaaS
Target audience: Website owners, digital marketers, SEO professionals, developers, agencies who want their content to be surfaced by AI models and LLM-powered search
Core features:
  - 0–100 AI Visibility Score: single composite score measuring how well a site is indexed by AI agents

  - Actionable fixes: per-issue code fixes with score impact and effort estimates (free tier: 1 fix; paid: all fixes)
  - Project management + scheduled weekly scans: monitor score over time for multiple sites
  - Shareable scan reports: public URLs for sharing AI visibility scores with clients or teams
  - API access (Agency tier): programmatic scan triggering via API key
