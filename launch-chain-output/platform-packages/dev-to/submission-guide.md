# dev.to — Submission Guide
1. Go to https://dev.to/new
2. Paste article.md content
3. Set canonical_url to: https://conduitscore.com/blog/ai-visibility-guide (publish that URL on your own domain first, or use the dev.to URL as canonical if posting there first)
4. Tags: seo, ai, webdev, tools
5. Set cover image (1000×420 px recommended)
6. Publish — no approval queue
7. Share in the DEV Community #discuss and #seo tags
