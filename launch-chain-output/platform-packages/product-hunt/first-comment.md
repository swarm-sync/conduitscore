# Product Hunt First Comment — Maker Post

Hey Product Hunt 👋

I built ConduitScore because I kept noticing something frustrating: sites that rank well in Google were completely invisible in ChatGPT and Perplexity results. I'd search for the same thing in both and get totally different sources cited.

The problem wasn't content quality — it was structural. Missing llms.txt files. Schema markup AI crawlers couldn't parse. robots.txt rules accidentally blocking AI bots. Issues invisible to standard SEO tools because those tools weren't built for this new reality.

**What ConduitScore does:**

Paste any URL → get a 0–100 score across 7 dimensions:
- **Citation signals** — author attribution, E-E-A-T, expertise markers
- **Content quality** — depth, readability, AI-parseable structure  
- **Content structure** — heading hierarchy, semantic HTML, answer patterns
- **Crawler access** — robots.txt, sitemap.xml, AI bot permissions
- **llms.txt** — presence and correct formatting of the AI-native crawl file
- **Structured data** — JSON-LD, schema.org markup
- **Technical health** — load speed, HTTPS, Core Web Vitals

Each issue comes with a specific code fix, a score impact estimate, and an effort rating so you know where to start first.

Free tier: 3 scans/month, 1 detailed fix per scan. Paid tiers unlock all fixes, scheduled monitoring, and API access for agencies.

I'd love to hear what you think — especially if you try it on your own site. The scores tend to be surprising. Most well-maintained sites still score below 50.

Drop your score in the comments. Happy to explain any specific finding.
