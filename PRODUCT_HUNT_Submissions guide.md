### A. Thumbnail
- [ ] 1 square thumbnail asset
- [ ] Use logo mark or product mark
- [ ] High contrast
- [ ] No tiny text
- [ ] Export at Product Hunt-safe dimensions

### B. Gallery
- [ ] Create at least 5 gallery images
- [ ] Gallery image 1 = homepage with scanner visible
- [ ] Gallery image 2 = score/results screen
- [ ] Gallery image 3 = issue breakdown
- [ ] Gallery image 4 = fixes/prioritization view
- [ ] Gallery image 5 = methodology / what is checked
- [ ] Add caption text for each gallery image
- [ ] Export all gallery images at Product Hunt-safe dimensions

### C. Demo media
- [ ] Create a short demo GIF or video
- [ ] Show: paste URL → run scan → view score → view issues → view fixes
- [ ] Keep short and easy to understand
- [ ] No audio required unless it clearly improves comprehension
