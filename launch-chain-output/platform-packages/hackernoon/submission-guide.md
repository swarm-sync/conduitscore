# HackerNoon — Submission Guide
1. Create account at https://hackernoon.com
2. Go to https://hackernoon.com/write
3. Paste article.md content
4. Add tags from frontmatter: seo, artificial-intelligence, web-development, chatgpt, machine-learning
5. Set canonical URL (your own blog if published there first)
6. Submit — HackerNoon has an editorial review queue (typically 2-5 business days)
7. Respond to any editorial feedback promptly
NOTE: HackerNoon prefers technical depth and original data. The scan statistics (median score 42/100, 85% missing llms.txt) are strong editorial hooks — make sure they are accurate before submitting.
