# Medium — Submission Guide
1. Go to https://medium.com/new-story
2. Paste article.md content (remove the markdown — Medium is WYSIWYG)
3. Set canonical URL in Settings → Advanced Settings → Canonical link (point to your own domain or leave blank if Medium is the first publish)
4. Add tags: SEO, Artificial Intelligence, Web Development, Content Marketing, SaaS
5. Submit to publications: "Better Marketing", "The Startup", "Towards AI" (search and request submission)
6. Set custom thumbnail image (1400×787 px recommended)
7. Publish
NOTE: Medium canonical chain — if you publish on your own blog first, set canonical_url here to your blog URL.
