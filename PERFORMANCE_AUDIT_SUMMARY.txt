================================================================================
  CRITICAL FINDINGS
================================================================================

1. RENDER-BLOCKING CSS: 150ms (11.7 KiB)
   Location: src/app/globals.css:2 (@import "tailwindcss")
   Impact: -600 to -800ms LCP delay
   Root Cause: Full Tailwind CSS loaded synchronously before paint
   Fix: Install Critters plugin for automatic critical CSS extraction
   Effort: LOW (1 line in next.config.ts)

2. LEGACY JAVASCRIPT & POLYFILLS: 14 KiB ES5 Code
   Location: package.json:55-62 (browserslist)
   Impact: -240 to -320ms LCP delay
   Root Cause: Targeting old browsers (chrome >= 90, safari >= 15)
   Fix: Update browserslist to modern browsers (chrome >= 100, safari >= 16)
   Effort: LOW (update 6 lines in package.json)

3. UNUSED JAVASCRIPT: 27.3 KiB Deferred
   Location: recharts (18 KiB), framer-motion (12 KiB)
   Impact: -180 to -240ms LCP delay
   Root Cause: Dashboard-only libraries loaded on all pages
   Fix: Lazy-load with Next.js dynamic() on dashboard only
   Effort: MEDIUM (30 minutes)

4. LOW CONTRAST BUTTON: 5.2:1 (AA, fails AAA)
   Location: src/components/scan/scan-form.tsx:339
   Button: "Scan My Site Now" (white text on red #ff2d55)
   Impact: WCAG 2.1 AA compliance failure
   Fix: Darken red to #E8004D (6.8:1 contrast ratio)
   Effort: LOW (1 line change)

5. LONG MAIN-THREAD TASK: 67ms Forced Reflow
   Location: src/components/scan/scan-form.tsx:349-360
   Issue: Inline onMouseEnter/onMouseLeave change boxShadow (layout prop)
   Impact: -10 to -15ms TBT delay
   Fix: Replace inline styles with CSS class (.scan-btn:hover)
   Effort: LOW (add CSS class, remove event handlers)
