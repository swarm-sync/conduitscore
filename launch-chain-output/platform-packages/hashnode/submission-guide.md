# Hashnode — Submission Guide
1. Create account at https://hashnode.com
2. Go to https://hashnode.com/create/story
3. Paste article.md content
4. Set canonical URL if published elsewhere first
5. Add tags: seo, ai, tools, webdev
6. Choose your blog (or Hashnode's general feed)
7. Publish — no editorial queue
8. Share in the Hashnode community Discord and relevant tags
